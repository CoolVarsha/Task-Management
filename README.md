Project Overview:

TaskMaster is a modern task management app designed to help users organize, track, and prioritize tasks efficiently. The app features a clean and responsive interface with real-time updates and smooth navigation.

Key Features:
Task CRUD: Create, read, update, delete tasks
Task Status: Mark tasks as pending, in-progress, or completed
Dashboard: Overview of tasks and progress
Performance Optimization: Lazy loading, memoized components
Responsive Design: Works seamlessly on desktop and mobile

Technologies Used:
Frontend: React.js, ES6, HTML5, CSS3, Tailwind CSS
State Management: Redux (with async actions)
Routing: React Router v6
